# redis-analytics

Reusable Redis analytics primitives for:

- RedisTimeSeries metrics
- HyperLogLog unique counts
- Bloom-filter backed first-seen tracking
- Query builders for grouped and dimensional views

## Why this package

- Centralized analytics primitives that can be shared across services
- Consistent time bucketing and timeframe logic
- Small API surface with composable stores and query builders
